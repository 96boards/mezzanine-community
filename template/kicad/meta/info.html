96Boards

Mezzanine Board Template

This is a project template for developing a 96Boards Mezzanine board (compatible with these boards).

This project's schematic only contains the necessary connectors needed for a mezzanine board. The current connectors are surface mounted, which can be easily changed to through hole. This is noted in the schematic.

The PCB includes the connectors, and mounting holes. The connectors and holes have been positioned so that the boards are aligned when mounted. The edge cuts also aligns with the base board. All the mezzanine board IO is connected to the header pins.
